===============
crm_usability_misc
===============

Various modification of CRM app

Installation
============

Use Odoo normal module installation procedure to install
``crm_usability_misc``.

Description
===========

- Hide the Campaign and Medium fields
- Hove “Source” and “Referred by” under Tags
- Move “Sales team” under Phone
